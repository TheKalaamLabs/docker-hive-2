# Apache Hive

Simple docker images for hive

# How to use

## Build Image

```bash
./build
```

## Config

Modify hack/set-default.sh.tmpl


## Run

```bash
# Start hiveserver and hivemetastore
hack/start_hive.sh

# Start hive shell
hack/hive
```
